
# Обновление документации

При добавлении или изменении функциональности, видной для пользователя:

1. Обновить PRD.md:

2. Обновить config.yaml:
   - Добавить новую функциональность в help_text
   - Обновить список возможностей бота

3. Подготовить пост для телеграм-канала бота об обновлении, если изменение будет заметно
для пользователей (админов).
